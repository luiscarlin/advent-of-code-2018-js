### Advent of Code 2018 JavaScript

My solutions to Advent of Code 2018 in JavaScript

## Get Started

```bash
# install dependencies
yarn install

# set some env vars
cat <<EOF >.env
COOKIE=your AoC cookie
YEAR=year of AoC
EOF

# get input for day 1 (or any other day)
node ./get_input.py 1 > 1.in
```
